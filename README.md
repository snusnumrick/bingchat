# bingchat
Console chat with Bing AI in node.js

## To Run
- `cp .env.example .env`
- In the edge browser login and open Bing Chat, locate Bing _U cookie and put its content into `.env`
- install node.js with version >= 18
- `npm install`
- `npm start`
